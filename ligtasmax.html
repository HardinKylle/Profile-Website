LigtasMax: Disaster Assistance System

A mobile and web application to enhance disaster response in Manila City using geofencing, spatial clustering, and real-time alerts.

My Contributions

  • Developed the responder-facing web application using React, creating a real-time dashboard for incident management.
  • Integrated the Google Maps API to visualize incoming incident reports, active geofences, and user-density clusters from the DBSCAN algorithm.
  • Collaborated with the Innovation Integrated GIS and Data Hub to design a dashboard that enables responders to validate reports and prioritize high-density areas.
