LigtasMax: Disaster Assistance System

Project Overview

LigtasMax is a comprehensive mobile and web application I helped develop as my thesis project. It was designed to enhance disaster response in Manila by providing real-time, location-based alerts and helping emergency services prioritize aid efforts.

Key Features

  • Real-time, community-based reporting for floods and fires.
  • Geofencing to automatically alert users within a disaster area.
  • Use of the DBSCAN clustering algorithm to analyze user density and identify hotspots for aid.
  • A visual dashboard for the GIS Manila Data Hub for monitoring and coordination.

Technology Stack

  • Firebase
  • DBSCAN Algorithm
  • Geofencing
  • GIS
  • Mobile (iOS & Android)
  • Web Application
  • Google API Integration
