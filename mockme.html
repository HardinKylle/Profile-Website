MockMe

A 3D device mockup web application that allows you to upload UI images/videos, view them in a 3D model, animate scene sequences, and capture visual assets for design showcases.

My Contributions

  • Developed a browser-based 3D product mockup generator where users can apply custom UI mock-ups to device models, modify background colors, and manipulate the scene using scene controls or preset camera views
  • Built a timeline-based animation editor allowing users to create custom camera movement tracks and set animation durations
  • Integrated Firebase (Auth and Firestore) to manage user accounts and enable saving/loading of user projects and assets
  • Implemented a high-resolution export feature, allowing users to capture their 3D animations as MP4 videos or images at various aspect ratios
